¿De qué se trata?

Acontinuación y a lo largo de este sitio web, se encontrará información referente a temas de física que creemos que son importantes para un desarrollo efectivo de conocimientos previos en el área de física, valga la redundancia, y así mismo brindar ayuda para poder apoyar a los usuarios para solucionar dudas referentes al áre anteriormente mencionada.

Además de ello, se cuenta con actividades evaluativas que permiten que el usuario pueda tener retroalimentación de los conocimientos que necesita reforzar para realmente asegurar un aprendizaje óptimo y adquirir buenas bases.
